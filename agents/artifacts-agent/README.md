# Artifacts Agent

Especialista en generacion de artefactos y documentacion tecnica.

## Archivos

- `master-prompt.md`: prompt maestro.
- `logic.ts`: logica base.
- `config.json`: configuracion.
- `behavior-rules.md`: reglas de comportamiento.
- `functions.md`: funciones disponibles.
- `internal-docs.md`: documentacion interna.
- `examples.md`: ejemplos de uso.

## Integracion GitHub Scanner

Puede recolectar ejemplos reales de archivos y plantillas open-source para acelerar la creacion de artefactos tecnicos.
