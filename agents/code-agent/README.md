# Code Agent

Especialista en analisis y cambios de codigo.

## Archivos

- `master-prompt.md`: prompt maestro.
- `logic.ts`: logica base.
- `config.json`: configuracion de capacidades.
- `behavior-rules.md`: reglas de comportamiento.
- `functions.md`: contrato de funciones disponibles.
- `internal-docs.md`: documentacion interna operativa.
- `examples.md`: ejemplos de uso.

## Integracion GitHub Scanner

Este agente puede solicitar ejemplos reales de codigo open-source via backend (`/api/github/examples`) para comparar patrones y proponer implementaciones mas solidas.
