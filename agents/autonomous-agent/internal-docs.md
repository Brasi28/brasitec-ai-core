# Documentacion Interna - Autonomous Agent

## Modelo de ejecucion

1. Parsear mision y restricciones.
2. Construir plan por hitos.
3. Delegar tareas a agentes especializados.
4. Verificar salidas y consolidar resultado.

## Criterios de parada

- Objetivo cumplido.
- Bloqueo externo no resoluble.
- Riesgo alto detectado.
