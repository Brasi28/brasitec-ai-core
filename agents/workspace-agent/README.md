# Workspace Agent

Especialista en lectura estructural y analisis del workspace.

## Archivos

- `master-prompt.md`: prompt maestro.
- `logic.ts`: logica base.
- `config.json`: configuracion de capacidades.

- `examples.md`: ejemplos de uso.
