# evolution-engine

Orquestador de evolucion continua por fases.

## Fases

- `turbo` (dias 1-5): aprendizaje 24h + limpieza nocturna.
- `stable` (dia 6 a mes 6): 3 ciclos al dia + limpieza semanal.
- `infinite` (mes 6+): evolucion indefinida con optimizacion mensual.

## Uso

```bash
npm --workspace tools/evolution-engine run run
```
